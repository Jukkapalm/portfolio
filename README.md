# Jukka Palm - Portfolio

AI-teemainen portfolio-sivusto.

---

## Rakenne

```
/           ← v1 (alkuperäinen versio)
/v2/        ← v2 (uusin versio)
```

> **Uusin versio löytyy kansiosta `/v2`**

---

## Teknologiat

- HTML
- CSS
- JavaScript
- Bootstrap

---

## Ominaisuudet

- AI-teemainen terminaali-UI
- Neuroverkko-partikkelianimaatio taustalla
- Interaktiivinen komentorivi (`ls projects/`, `open unit_01`, `help`)
- Projektikorttien hover-laajennus
- Projekti-popup kuvan, kuvauksen ja linkkien kanssa
- Käynnistysanimaatio flickerillä
- Responsiivinen layout (mobiili + desktop)

---

## Komentoriviohjeet

| Komento | Toiminto |
|---|---|
| `ls projects/` | Listaa kaikki projektit |
| `open unit_01` | Avaa projekti |
| `cat unit_01` | Avaa projekti |
| `./unit_01` | Avaa projekti |
| `clear` | Tyhjennä output |
| `help` | Näytä ohjeet |
